# Upstream implementation

The initial extension implementation is derived from Floating Mini Panel v8 by
Gerhard Himmel:

- UUID: `floating-mini-panel@ghi59.github.com`
- upstream: <https://github.com/ghi59/floating-mini-panel>
- license headers in the source specify GPL-2.0-or-later;
- local baseline was the installed, GNOME Shell 50-compatible patched build.

Local changes introduce configuration-driven plugins, role filtering, compact
styling and the CPU-load monitor. Preserve upstream copyright/license headers
when modifying derived files.
