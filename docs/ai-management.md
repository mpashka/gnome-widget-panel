# AI-assisted panel management

Press a button, describe a desired indicator, then let an external AI
orchestrator search registries, inspect provenance and permissions, prefer an
existing widget, or prepare a reviewed patch. It runs tests in a sandbox,
displays the diff, installs only after confirmation, and collects screenshot/log
evidence.

Voice input must never directly execute downloaded GJS inside GNOME Shell.
Pin revisions, record provenance, require explicit permission approval, and keep
rollback available.
