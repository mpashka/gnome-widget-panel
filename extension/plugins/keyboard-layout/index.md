# keyboard-layout widget

Back to [plugins index](../index.md).

## Purpose

Shows the GNOME keyboard layout indicator inside the floating panel.

## Source files

- `index.ts` — creates an `IndicatorsDrawer` filtered to the `keyboard` role and
  forces the indicator into the always-visible area.
