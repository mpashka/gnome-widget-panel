# Extension source index

TypeScript source tree and static assets for the GNOME Shell extension. This is
the source of truth; `../extension/` is generated by `npm run build`.

## Directories

- [`plugins/`](plugins/index.md) — built-in panel widgets, one directory per
  widget.
- `config/` — bundled default widget order and enabled-state configuration.
- `schemas/` — GSettings schema copied into generated extension and compiled by
  `install.sh`.

## Core modules

- `extension.ts` — GNOME Shell extension entrypoint and floating panel actor.
- `pluginManager.ts` — reads widget config and instantiates plugin actors.
- `controlButton.ts` — panel handle, menu and movement controls.
- `indicatorsDrawer.ts` — reusable GNOME indicator clone drawer.
- `utils.ts` — monitor/panel helper functions.

See [`../docs/object-model.md`](../docs/object-model.md) for runtime objects.
